About This Page

Layout

This page was built using a hybrid of Bootstrap and from-scratch techniques.

Media queries were used to achieve the minor responsiveness, while containers, rows, and columns were provided from Bootstrap.

Bootstrap also provides the offcanvas document used to display this page info.

Functionality

This page uses Javascript to provide interactivity.

DOM manipulation is used here to change the background image and music based on the search results.
